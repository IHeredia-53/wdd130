General Experience and Reflections

The most challenging part of this project was the CSS as the lead developer. I was responsible for forming CSS that would be sufficiently adequate for what the client requested. As a junior developer, I had to make minor adjustments to the already-developed CSS file for my own contributions, but I did not have that privilege as the lead developer. I enjoyed working with GitHub and forming (or helping to form) websites. If I could do this project again differently, I would be more specific as to how I wanted my website. I'm content with how it turned out, but I know that I will make some of my own changes to it. This was a great introduction into how real-world projects could look like, and I believe I'm ready to help with more projects in the future.
